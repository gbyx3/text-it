# Installation
### Twilio
https://support.twilio.com/hc/en-us/articles/223136027-Auth-Tokens-and-How-to-Change-Them  

### Docker
```
git clone https://github.com/gbyx3/text-it
cd ./text-it
docker build -t text-it:latest .
docker run --rm -p 3000:3000 text-it
```
